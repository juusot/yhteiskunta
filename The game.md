The game needs the following structure:

1. Buildings
2. Resources
3. Characters
4. Biome/nature/environment
5. Items (what characters use)
6. Groups
7. Vehicles (what characters can use to move stuff/characters)

# 1. Buildings

The game needs specific places where characters hold their stuff, live and where the groups/nations etc start to form up. If there are no buildings then there cannot be any civilization. At the start of the game there is one or more characters who start to gather resources around them to start building things, like warehouse. Warehouse makes it possible to form these societies.

# 2. Resources

All resources that are being gathered should deplete eventually, but also there should be some regeneration loop, so people won't starve or run out of things. For now let's focus on 3 resources: wood (for buildings), gold (for money), field and bushes (for food). Wood, gold and bushes simply disappear after X amount of resources has been gathered and then respawn randomly nearby (but not on top of anything). Fields however should be built by characters. Those will never disappear, unless they take damage and get destroyed.

# 3. Characters

The main driving force in the game. No characters, no action. The environment stays rather static (at least for now). We will eventually add different kinds of characters (like animals, robots etc), but for now let's just focus on "humans". Characters can own stuff, carry stuff and do actions.

# 4. Environment

Have different features. River/shallow water is swimmable, but deep water (ocean) only by boats. For now let's just have river/shallow water, grass and mountains. Mountains are impassable, unless someone goes over them by vehicle.

# 5. Items

Tools, armor, weapon, money etc. All of these should be made with resources, can be stored, destroyed used etc. Can also give stats for Characters, like groups.

# 6. Groups

The main organizational force behind the game. Characters can belong to multiple groups and groups will have hierarchy. For example a cult member hierarchy is: Cult, Nation, Clan, but normal citizen is Nation, Clan. Meaning if there is ever conflict between groups higher hierarchy wins. Groups also gives stats to Characters and will be stacking if needed.

# 7. Vehicles

Like items, can be built, stored, used, destroyed. The main thing is that vehicle will transport one or more Characters.
